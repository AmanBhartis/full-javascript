Lecture 2

this is about alert and prompt and confirm functions in JavaScript.
alert is used to show a message to the user.
prompt is used to get input from the user.
confirm is used to get a yes or no answer from the user.
we show an alert message to the user and then we ask the user for their name using prompt and then we show a confirm message to the user asking if they want to see a greeting message. If the user clicks on OK, we show a greeting message using alert. If the user clicks on Cancel, we show a goodbye message using alert.

we save the confirm response in a variable and then we check if the response is true or false. If the response is true, we show a greeting message using alert. If the response is false, we show a goodbye message using alert by using an if statement.
